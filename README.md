# Talend Auditor

Ce programme à pour objectif de ressortir des métriques et des altertes concernant les Job TALEND.

Il a été éprouvé pour les versions de Talend 5.4.1 et 6.3.1.
Sur des versions 3 et 4, certaines métriques ne fonctionnent pas.

Pour exécuter le programme, simplement passer en argument le chemin vers le zip d'export du projet Talend ou la racine du projet dans le workspace Talend.
